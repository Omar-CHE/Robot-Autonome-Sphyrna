# 🦈 Projet Sphyrna : Robot Suiveur de Ligne Autonome & Simulateur

Bienvenue sur le dépôt du projet Sphyrna, développé dans le cadre de mon cycle préparatoire à l'ENSAM Rabat. L'objectif était de concevoir un système embarqué complet (matériel et logiciel) capable de navigation autonome.

## 🎯 Fonctionnalités clés
- **Régulation PID :** Développement de l'algorithme de contrôle en C++ pour stabiliser la trajectoire à haute vitesse.
- **Simulation Prédictive :** Création d'un environnement virtuel sur-mesure pour tester la logique algorithmique avant le déploiement matériel.
- **Ingénierie Mécanique :** Modélisation CAO d'un châssis ultra-compact (150mm) sous Autodesk Fusion 360.

## 💻 Architecture du Dépôt
* `/Code_Embarque` : Contient le code source C++ (IDE Arduino).
* `/Simulateur` : Scripts et assets liés à l'environnement de simulation.
* `/Modelisation_3D` : Rendus du châssis.
* `/Documentation` : Rapport technique détaillé du projet.

## 🛠️ Technologies & Outils
* **Langages :** C++, Python (Pygame)
* **Matériel :** Microcontrôleur Arduino, capteurs TCRT5000, drivers L298N
* **Logiciels :** Autodesk Fusion 360, Notion (Gestion Agile)
